# 主播数据监测看板使用说明

## 每天怎么用

1. 把每天导出的 `主播概览列表_*.csv` 放在这个文件夹，或放进 `daily` 文件夹。
2. 双击 `启动看板.bat`。
3. 浏览器打开 `http://localhost:3000` 后查看部门总览、经纪人进度、预警名单和主播明细。

## 线上多人查看

线上地址：

- `https://anchor-task-dashboard.netlify.app/`

普通成员直接打开线上地址查看。管理员更新数据时：

1. 打开线上地址。
2. 展开 `上传 / 更新数据`。
3. 输入 Netlify 环境变量 `ADMIN_PASSWORD` 对应的上传密码。
4. 上传每日 `主播概览列表_*.csv`。
5. 月初或任务调整时，同时上传 `月任务.csv` 和 `姓名对应.csv`。
6. 上传完成后页面会自动刷新，所有人看到同一份最新数据。

## Spaceship 域名接入

在 Netlify 里把你的域名添加到 `anchor-task-dashboard` 站点后，再到 Spaceship 的 DNS 里设置：

- `www`：添加 `CNAME`，指向 `anchor-task-dashboard.netlify.app`
- 根域名 `@`：按 Netlify 给出的 A 记录或 ANAME/ALIAS 提示填写

DNS 生效后，Netlify 会自动申请 HTTPS 证书。

## 每月怎么维护任务

打开 `月任务.csv`，维护这些列：

- `月份`：格式用 `2026-06`
- `经纪人`：任务表里的姓名
- `入职时间`
- `入职月数`
- `上月流水`
- `目标总流水`

## 姓名不一致怎么处理

打开 `姓名对应.csv`，把任务表姓名和日报里的运营经纪人名称对应起来。

例子：

- `潘亚文,潘潘`
- `张可,可可`

如果看板提示“有目标无日报”或“缺少目标”，通常就是这里还没补对应关系。

## 当前口径

- 流水归属按日报里的 `运营经纪人`
- `总音浪` 按当月累计流水计算
- 月进度按日报文件日期到月底计算
- 主播有效数按 `语音有效天 > 0` 计算
- 没有运营经纪人名字的主播不计入数据
- `念卿` 和念卿名下主播不计入数据
- 上次开播时间达到 `1月前`、更久或 `从未开播` 的主播不计入看板数据
- 上次开播时间达到 `2日前` 起，在主播栏中标红提醒
